URL Shortener Microservice

User Stories

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • If I pass an invalid URL that doesn't follow the valid http://www.example.com format, the JSON response will contain an error instead.
  • When I visit that shortened URL, it will redirect me to my original link.

Usage

You can create a new short url for https://www.somesuperlong.link/that-nobody/would-want?to=type by visiting https://vesper-fcc-api.glitch.me/url-shortener/new/https://www.somesuperlong.link/that-nobody/would-want?to=type .

That will return something like { "original_url":"https://www.somesuperlong.link/that-nobody/would-want?to=type", "short_url":"https://vesper-fcc-api.glitch.me/url-shortener/2" }

Then, visiting https://vesper-fcc-api.glitch.me/url-shortener/2 would redirect to https://www.somesuperlong.link/that-nobody/would-want?to=type
